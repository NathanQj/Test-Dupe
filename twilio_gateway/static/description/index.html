Twilio SMS Gateway

This application works with SMS Notification application.

Basic Features
  • Supporting application for "SMS Notification" module. Means this module is an add-on of Module "SMS Notification" . To use this module you must have installed "SMS Notification" first.
  • This application provide required SMS gateway facility to "SMS Notification" application for sending notification to different mobile number.
  • Twilio Gateway module is able for sending text SMS directly to the sellers,suppliers,buyers,followers,partners,customers etc.
  • Ability to send message when product order is placed by admin.
