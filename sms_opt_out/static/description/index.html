Odoo SMS Opt-Out

Odoo SMS Alerts Opt-Out Option In Customer’s Account


Give Freedom to customers to Opt-Out from Odoo SMS Alerts and Notifications.

Added Odoo SMS Opt-Out Portal In Customer’s Account.

A Customer-friendly and convenient way to manage SMS alerts for various customers.

The module works in conjunction with Odoo SMS Notification Module to provide the SMS alerts and unsubscribe features to your website customers.

Select Which SMS types to Opt-Out From in Website Portal


It Facilitates the customers to select which type of Odoo SMS alerts they wish to unsubscribe from.

Allow customers to disable one or more Odoo SMS alerts such as promotional SMS, order SMS alerts, SMS notifications related orders, invoices, etc. from the list.

As more SMS notification templates are added in Odoo backend, new options appear in the customer’s account in the Odoo SMS Alerts Opt-out List.

Unsubscribe Customers From Odoo SMS alerts From Odoo Backend


Added option for Odoo User to manually Unsubscribe Odoo customers from SMS Alerts from the backend.

Manage SMS Alerts for individual customers separately from backend.

You can also hide the Odoo SMS Opt-Out option for customers completely from backend.

Multi-Language Compatible


The Odoo SMS Opt-Out portal is Odoo Website Language compatible.

It allows the customer to change the language of the portal from the Odoo Website.

Enable/Disable SMS Opt Out Option

Use Configuration settings to show/hide SMS Opt-Out option for Customer. It will hide the option for all customers.

(However, disabling the option will not affect changes already made by old customers.
